# scripts/

This folder contains **developer helper scripts** for setup, running, and maintenance.

## Planned Scripts

| Script | Purpose |
|--------|---------|
| `setup_python_env.bat` | Create Python virtual environment and install dependencies |
| `setup_node.bat` | Install Node dependencies for Electron + React |
| `run_engine.bat` | Start the Python FastAPI backend manually |
| `run_dev.bat` | Start full dev environment (backend + Electron) |
| `reset_db.bat` | Delete and recreate the local SQLite database |
| `run_tests.bat` | Run all Python tests |

> Scripts will be added as each component is set up.
